首先戳右边那个download zip， 下载一个程序包~~请务必解压在一个你喜欢的地方
然后请打开命令行（LaunchPad里面搜索Terminal就好啦）
点开以后请输入如下神奇代码

sudo easy_install pip (按一下回车)
sudo pip install lxml  (按一下回车)
sudo pip install pillow  (按一下回车)
sudo pip install numpy  (按一下回车)
sudo pip install requests  (按一下回车)

以上各步都需要输入密码，嗯，输密码的时候没有字符显示出来，请盲打
如果安装不成功请咨询身边的程序员
